PHASMO TIMER
Phasmo-Timer is a lightweight transparent overlay utility built for Phasmophobia investigators.

This overlay uses AutoHotkey (AHK) and includes:

• Always-on-top overlay
• Smart ghost hunt timer
• Multiple overlay visibility modes
• Global hotkeys
• Lightweight standalone EXE
• Fully open-source source code

Phasmo-Timer does not read game memory, inject into the game, modify files, or bypass anti-cheat systems. Please check the source code before downloading the exe file for peace of mind. All files are hosted on GitHub.

It is simply a transparent overlay utility with global hotkeys and timer functionality.

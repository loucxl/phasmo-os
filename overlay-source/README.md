# Phasmo-Timer Overlay

Phasmo-Timer is a lightweight transparent overlay utility for Phasmophobia.

This project:
- does NOT read game memory
- does NOT modify game files
- does NOT inject into the game
- does NOT bypass anti-cheat systems
- does NOT collect user data

It is simply a transparent timer overlay with global hotkeys.

---

# Overlay Features

- Always-on-top overlay
- Smart ghost hunt timer
- Global hotkeys
- Draggable overlay
- Multiple overlay modes
- Lightweight design
- Open source

# Want To Compile It Yourself?

You can compile the overlay yourself instead of using the provided EXE.

1. Install AutoHotkey v2
2. Open Ahk2Exe
3. Select:
   - overlay.ahk
   - icon.ico
4. Compile:
   - Phasmo-Timer.exe

This allows users to inspect, modify, and compile the overlay themselves if preferred.

# Hotkeys

F6 = Start/Pause

F7 = Reset

F8 = Overlay Mode

F10 = Exit Overlay
